Projekat za tutoring sistem
Aplikacija koja koristi  REACT+SPRINGBOOT+MYSQL stack i koja predstavlja način implementacije
jednog tutoring sistema. Aplikacija je također fokusirana na savrmeneni pristup 
održavanja sesije prilikom čega koristi JSON Web Tokens koji su manuelno implementirani
s obzirom na činjenicu da Spring sam po sebi ne podržava JWT out of the gate.
Omogućava razne funkcionalnosti uključujući DM funkcionalnost, funkcionalnost kreiranja i pregledavanja predavanja,
postavljanje assignmenta...

Kako pokrenuti
Projekat ima dva režima rada , localhost  i remote koji omogućava pokretanje servera (react kao i spring boot) 
na lokalnoj wifi mreži koja poslije omogućava lakši deploy na pravi remote server , ali također aplikacija 
omogućava (na dev strani) lakšu izmjenu između ovog i development režima rada
